# ALGORITMO-DE-EUCLIDES-EXTENDIDO
-Para ejecutar el programa solamente se deben ingresar 2 números (ambos positivos y enteros).

-EJEMPLO: 
    EuclidesE(24,18) -----> mcd= 6 , Valor de x=1 , Valor de y= -1
    *Donde: donde mcd(a, b) = ax + by (x, y ∈ Z)
    Por lo tanto:
                mcd(24,18) = 24 (1) + 18 (-1)   
    Obtenemos:   6.
